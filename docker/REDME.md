# Database Structure

## Table1 - beat_files


| name     | type    | comments       |
|----------|---------|----------------|
| id       | integer | primary key    |
| name     | varchar | file name      |
| filetype | varchar | file type name |

## Table2 - users

| name          | type    | comments           |
|---------------|---------|--------------------|
| id            | integer | primary key        |
| name          | varchar | user Full name     |
| date of birth | varchar | user date of birth |
| email         | varchar | user email         |
